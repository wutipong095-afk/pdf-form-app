# แพ็กฟอร์มโรงเรียน v1

เทมเพลตตำแหน่งฟิลด์เริ่มต้นสำหรับงานที่พบบ่อย:

| ไฟล์ | ใช้กับ PDF (แนะนำชื่อ) |
|------|------------------------|
| `ใบเบิก.json` | `ใบเบิก.pdf` |
| `ใบจัดซื้อ.json` | `ใบจัดซื้อ.pdf` |
| `ใบขออนุญาตเดินทาง.json` | `ใบขออนุญาตเดินทาง.pdf` |

พิกัดเป็นจุดเริ่มต้น — โรงเรียนควรเปิดโหมดมาร์คจุดแล้วจูนให้ตรงฟอร์มจริง

ติดตั้งจากแอป: ปุ่ม **ติดตั้งแพ็กฟอร์ม v1** หรือ `POST /api/formpack/install`
