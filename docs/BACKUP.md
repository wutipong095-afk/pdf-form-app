# สำรองและกู้คืนข้อมูล

## สำรอง

ในแอปกด **สำรองข้อมูล** จะได้ไฟล์ `pdfmarker-backup-….zip` ประกอบด้วย:

- PDF ต้นฉบับที่อัปโหลด (`uploads/`)
- เทมเพลต (`templates_json/`)
- PDF ที่กรอกแล้ว (`output/`)
- ค่าตั้งคลังเอกสาร (`library.json` + `.pdfmarker` ถ้ามี)

**ไม่มี** `machine_id` / `license.json` / `secret_key` / ไฟล์ log

## กู้คืน

1. กด **กู้คืน** แล้วเลือกไฟล์ ZIP
2. เลือกโหมด:
   - **รวม (merge)** — ไม่ทับไฟล์ที่มีอยู่แล้ว
   - **แทนที่ (replace)** — ล้าง uploads/templates/output ของผู้ใช้แล้วแตกจาก ZIP

## ย้ายเครื่องใหม่

1. สำรองจากเครื่องเก่า
2. กู้บนเครื่องใหม่
3. **ขอคีย์ไลเซนต์ใหม่** จากผู้ขายด้วยรหัสเครื่องใหม่ — อย่าคัดลอก `machine_id` / `license.json` จากเครื่องเก่า

## เทมเพลตเดี่ยว

- **ส่งออกเทมเพลต** — ดาวน์โหลด JSON ของเทมเพลตที่เลือก
- **นำเข้าเทมเพลต** — อัปโหลดไฟล์ `.json` / `.tpl.json`

## แพ็กฟอร์ม v1

ติดตั้งเทมเพลตใบเบิก / จัดซื้อ / เดินทาง จากปุ่มในแอป แล้วจูนจุดมาร์คให้ตรงฟอร์มโรงเรียน
